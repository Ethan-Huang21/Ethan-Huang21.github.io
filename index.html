Portfolio

YouChat

A distributed Chat Application designed to exhibit fault tolerance and synchronization across multiple Pocketbase databases. Utilizes leader election and a FIFO queue with acknowledgments to ensure synchronization. Failure is masked in the event of replica, leader, and middleware failures.

Autocotrans

A Python script that utilizes the Selenium library to webscrape Cotrans to simplify mass file-upload and download. Utilizes a headless chrome browser to automate the entire translation process, keeping everything in the background.

OpenGL Demo

An interactive project that utilizes the C++ OpenGL Library to create and visualize Bezier and B-Spline curves. From these created curves, a 3D Model can be created and interacted with through camera movements. Lastly, a 3D Bezier Surface viewer that demonstrates other modes of visualization within computer graphics.

Python Basics

A project created within the Unity Engine over the course of 24 hours for a Hackathon. The game is a simple 3D Puzzle Platformer designed to teach Python terminology, syntax, and semantics. Awarded first place within the beginners stream that was competed in.

Omnifood

A project done following alongside Jonas Schmedtmann for a UDemy course based around the fundamentals of HTML and CSS. The website is designed to be responsive through design and media queries, ensuring that it's operable across different devices.
